# ⚡️ SwipeScore

SwipeScore is an **AI-powered content pipeline** that turns live sports data into engaging, "Stories-style" cards with automated quality checks. It consists of a powerful **n8n backend workflow** and a premium, dark-mode **React dashboard** for visualizing the results.

## 🚀 Features

- **Automated Data Ingestion:** Fetches the latest English Premier League match results from TheSportsDB automatically.
- **AI Story Generation:** Uses Google Gemini 1.5 Flash to synthesize match statistics into punchy, exciting 2-3 sentence stories, generating hashtags and suggesting background image queries.
- **Automated QA Checking:** A second Gemini AI agent fact-checks the generated story against the raw match data to ensure no hallucinations or incorrect scores, grading it on a 0-100 scale.
- **Beautiful React Dashboard:** A dynamic, glassmorphic UI built with Tailwind CSS. It features a feed of generated stories alongside an interactive sidebar detailing raw source data and QA issues.

## 🏗️ Tech Stack

- **Workflow Automation:** [n8n](https://n8n.io/)
- **LLM / AI:** Google Gemini (1.5 Flash) via native LangChain nodes
- **Frontend Framework:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS v3
- **Icons:** Lucide React

## 🛠️ Getting Started

### 1. n8n Backend Setup
The backend runs completely out-of-the-box using the provided workflow configuration.

1. Open your local or cloud **n8n** instance.
2. Go to your Workflows, click the menu in the top right, and select **Import from File**.
3. Select the `workflow.json` file located in the root of this repository.
4. Once imported, click on the two **Gemini Model (Story/QA)** sub-nodes and select your Google Gemini API credentials from the dropdown. 
5. Run the workflow. This will fetch new matches, run the AI chains, and output the compiled feed into `/public/data/stories.json`.

### 2. Frontend Dashboard Setup
The frontend consumes the JSON output generated by the n8n pipeline.

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser to `http://localhost:5173/` to witness your live AI content pipeline!
